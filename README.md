# Raspberry Pi @ home
我が家のRaspberry Piで動いている奴等です。

## script
- crontab.bk
  - cronのバックアップ。
- takePhoto.sh
  - 写真を撮ってSlackに投げ込むSript。
- time_report.sh
  - 時報をお知らせするScript。

### util
- cap.py
  - Raspberry Piに接続されているカメラで写真を撮る。
- fileUpload2Slack.py
  - ファイルをSlackにアップロード。
- jsay
  - 指定された文章を喋る。
